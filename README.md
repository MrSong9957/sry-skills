# sry-skills

技能驱动型工程自动化系统，基于Claude Code超能力工作流，实现从需求到交付的全链路自动化。

## 项目介绍

sry-skills是一个运行在严格项目结构下的工程自动化系统，负责解析用户意图、匹配技能、执行多阶段工作流、调度agents/commands/hooks/lib，并复刻Claude Code + 超能力工作流的完整工程流水线。

## 核心功能

- **技能驱动**：基于技能索引匹配用户需求，执行相应工作流
- **上下文感知**：提取最小必要上下文（MNC），确保任务执行的准确性
- **全程验证**：每一步执行后自动验证输出，确保结果可靠
- **可审计**：完整记录执行过程，便于追溯和分析
- **多技能支持**：包含代码简化、错误修复、前端设计、UI/UX优化等多种技能

## 主要技能模块

- **code-simplifier**：简化代码结构，提高可读性和性能
- **error-fix**：自动识别和修复代码错误
- **failure-analysis-taskforce**：分析系统故障，提供解决方案
- **frontend-design**：前端设计和实现
- **json-canvas**：JSON Canvas相关功能
- **summary-specialist-taskforce**：生成任务执行总结
- **ui-ux-pro-max**：高级UI/UX设计和优化
- **web-artifacts-builder**：构建Web相关制品
- **webapp-testing**：Web应用测试

## 工作流程

1. **启动阶段**：扫描技能，构建技能索引
2. **用户输入解析**：解析用户意图，识别任务类型和约束条件
3. **技能匹配**：基于技能索引执行语义匹配，选择合适的技能
4. **最小必要上下文（MNC）提取**：收集任务所需的最小上下文
5. **智能体选择**：根据任务阶段选择合适的智能体
6. **超能力工作流执行**：按顺序执行技能链
7. **验证与回滚**：每一步执行后验证输出，失败时执行回滚

## 技能定义格式

每个技能都包含以下部分：

1. **触发条件**：何时触发该技能
2. **输入**：技能所需的输入参数
3. **输出**：技能执行后的输出结果
4. **前置条件**：执行技能前必须满足的条件
5. **项目结构约束**：技能执行时的结构约束
6. **执行步骤**：技能的具体执行流程
7. **回滚策略**：失败时的回滚机制
8. **生命周期钩子**：技能执行前后的钩子
9. **参考文档**：相关参考资料
10. **示例**：技能使用示例
11. **核心目标**：技能的核心目标
